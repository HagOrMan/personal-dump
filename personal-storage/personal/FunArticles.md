- [Why AI won't render software dev obsolete](https://www.wsj.com/finance/softwares-death-by-ai-has-been-greatly-exaggerated-b639c0cd?st=iEaBRS&reflink=desktopwebshare_permalink). This is an interesting one because people have talked about it for years as AI capabilities have grown. We're now almost a year since this was written. I have and always will think that software dev at its core will never be obsolete, and I believe that sentiment is widely shared. I do believe that it will change, same as people moved from punching cards to run programs to using visually rich IDEs. Things change but the need to make software work does not.
